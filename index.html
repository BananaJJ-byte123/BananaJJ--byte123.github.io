Host Scheduling Assistant

Feishu Base + Vercel API + bot notifications

This is the online workspace version. Pages are deployed on Vercel, while schedule, host, room, and request data are read from and written to Feishu Base.

Manager Console

For supervisors to review requests, generate schedules, clean old records, and check monthly workload.

  • Read hosts, livestream rooms, shift templates, and requests
  • Generate schedules by date range or by a single day
  • Create manual assignments for a selected host and room
  • Track pending, approved, rejected, and historical records

Host Portal

For hosts to view their schedule, confirm shifts, submit swap requests, and add flexible availability.

  • Search and select your own host name
  • View personal upcoming schedule
  • Request shift swaps or cover by another available host
  • Add extra available time by date and shift slot
