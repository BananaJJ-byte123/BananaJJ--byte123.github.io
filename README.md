# Overseas Anchor Scheduler MVP

GitHub Pages static H5 build for the overseas anchor scheduling MVP.

## Entrances

- `index.html`: landing page with supervisor and anchor entrances
- `manager.html`: supervisor / admin console
- `anchor.html`: anchor self-service page

## Notes

This version is a static MVP. Data is stored in browser `localStorage`, so it is suitable for workflow demos and UI validation.

For real multi-user use, connect it to Feishu Base, Miaoda database, or a backend database.
